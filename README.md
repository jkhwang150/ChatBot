## 1. 목표와 기능
### 1-1. 목표
- 영화에 대한 견해를 나눌 수 있는 블로그
- 영화 추천이 가능한 블로그
- 최신 영화 소식을 접할 수 있는 블로그
### 1-2. 기능
- CRUD가 가능한 게시판
- ChatGPT API를 활용한 영화 추천을 받을 수 있는 블로그
- 회원가입, 로그인, 로그아웃 기능
- 로그인한 사용자에게만 CUD 기능 권한
- 게시글에 대한 댓글 등록(로그인한 사용자만 가능)
